# LABORATORIO 6: Simulación y monitoreo de variables cardiovasculares y hemodinámicas
## INTRODUCCIÓN
El monitoreo de signos vitales es fundamental en el ámbito clínico porque permite evaluar de forma continua el estado cardiovascular y respiratorio del paciente, identificar alteraciones fisiológicas y activar alarmas ante condiciones de riesgo. En este contexto, el monitor uMEC 100 permite registrar variables como la saturación periférica de oxígeno y la frecuencia cardíaca o de pulso, mientras que el simulador Pronk OxSim OX-1 permite recrear señales controladas de pulsioximetría, incluyendo diferentes valores de SpO₂, frecuencia de pulso y condiciones de baja perfusión. La integración de ambos equipos en la práctica de laboratorio permite verificar el funcionamiento del monitor, analizar la respuesta ante escenarios como bradicardia, taquicardia e hipoxemia, y relacionar la medición instrumental con criterios clínicos de seguridad. Por ello, esta revisión bibliográfica busca reunir información técnica confiable sobre el modo de operación del uMEC 100, las variables fisiológicas simuladas por el OxSim OX-1 y los errores máximos permitidos para cada parámetro, con el fin de sustentar adecuadamente la evaluación funcional del sistema de monitoreo.
## OBJETIVOS
### General 
Operar con el simulador Pronk OxSim (OX-1) y el monitor de signos vitales uMEC 100 para pruebas funcionales.
### Específicos
• Identificar los modos de operación de un simulador de parámetros hemodinámicos (Pronk OxSim).
• Verificar los límites de medición de un monitor de signos vitales mediante simulación de variables hemodinámicas.
• Interpretar variaciones en los parámetros hemodinámicos asociadas a estados fisiológicos y patológicos.
